About this deck

This page embeds the official Twitch and YouTube players directly — there are no ads, trackers, or analytics of its own here.

Ads that appear inside a Twitch or YouTube player come from Twitch/YouTube's own servers, same as on twitch.tv or youtube.com. Whether an extension blocks them depends entirely on that extension's own filter lists. The "Ad Shield" toggle in the sidebar keeps new/reloaded tiles muted by default so ad audio doesn't surprise you, but it can't remove the ads themselves — no page-level script can reach inside an official cross-origin player to do that.

EMBED PARENT DOMAIN

If a Twitch tile shows a grey "parent" error, this page is running somewhere Twitch doesn't recognize as a real host (e.g. opened directly as a file://). Serve it from any domain, or a local dev server like python3 -m http.server, and it'll resolve.

Keyboard: 1–9 focus a tile · a add source · m mute all · f theatre mode · Ctrl+B toggle sidebar · Esc back out.
