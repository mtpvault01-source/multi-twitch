About this deck

This page embeds the official Twitch and YouTube players directly — there are no ads, trackers, or analytics of its own here.

Ads that appear inside a Twitch or YouTube player come from Twitch/YouTube's own servers, same as on twitch.tv or youtube.com. Whether an extension blocks them depends entirely on that extension's own filter lists — this page doesn't need any special integration for that to work, since it's using the same official embeds those extensions already target.

EMBED PARENT DOMAIN

If a Twitch tile shows a grey "parent" error, this page is running somewhere Twitch doesn't recognize as a real host (e.g. opened directly as a file://). Serve it from any domain, or a local dev server like python3 -m http.server, and it'll resolve.
